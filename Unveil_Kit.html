Guide pas à pas
1
Entrez votre clé API Anthropic
En haut à droite. Votre clé commence par sk-ant-api03-. Elle n'est jamais stockée ni transmise — uniquement en mémoire session. Pour obtenir une clé : console.anthropic.com → API Keys. Coût estimé : moins de 0,10 EUR par candidature complète.
2
Préparez votre CV
Utilisez le modèle CV Word Unveil fourni dans le kit. Remplissez avec vos informations — remplacez tous les textes en italique gris. Supprimez les sections inutiles. Le modèle est optimisé ATS : pas de colonnes, pas de tableaux, Calibri 10pt.
3
Analysez votre profil
Onglet Mon Cavalier → collez le texte de votre CV ou uploadez une capture → cliquez Analyser. L'IA détecte vos forces et suggère des cibles prioritaires.
4
Scannez une annonce
Onglet Scanner → collez le texte de l'offre OU uploadez une capture d'écran (drag & drop supporté). Choisissez un angle. Lancez jusqu'à 5 scans sous des angles différents. Comparez et sélectionnez le meilleur.
5
Téléchargez votre CV + LDM
Onglet CV Optimisé → votre CV refaçonné et votre lettre de motivation sont prêts. Téléchargez en Word pour modifier, ou en PDF pour envoyer directement.
6
Nouvelle annonce → recommencez
Votre CV reste en mémoire. Collez une nouvelle annonce et relancez. Sauvegardez chaque candidature dans l'onglet Mes candidatures pour suivre vos envois.
FAQ
Qu'est-ce qu'un ATS ? +
ATS = Applicant Tracking System. Logiciel qui filtre automatiquement les CV avant toute lecture humaine. Il scanne les mots-clés, évalue la structure, attribue un score. 88% des candidats qualifiés sont éliminés avant d'être lus.
Pourquoi mon CV ne passe pas ? +
Causes fréquentes : mots-clés manquants (l'ATS cherche les termes exacts de l'annonce), format complexe (colonnes, tableaux, images illisibles), traces IA détectables. Unveil résout les trois.
Unveil garantit-il un emploi ? +
Non. Unveil augmente de façon considérable les chances que votre candidature soit enfin vue par un humain. La décision finale appartient toujours au recruteur.
Ma clé API est-elle sécurisée ? +
Votre clé n'est jamais stockée ni transmise. Elle reste uniquement en mémoire de votre navigateur pendant la session et disparaît à la fermeture de l'onglet.
Quelles polices sont compatibles ATS ? +
Polices sûres : Calibri, Arial, Times New Roman, Georgia, Verdana. À éviter : Futura, Gotham, SF Pro, Helvetica Neue Light. Le modèle Unveil utilise Calibri — ne changez pas la police.
Combien coûte une utilisation ? +
L'outil est gratuit. Vous payez uniquement les appels API Anthropic. Environ 0,05 à 0,10 EUR par candidature complète.
